# AI Showcase

This project is a collection of outputs from various AI models (GPT-4o, Claude 3.5 Sonnet, Gemini 1.5 Pro) across different tasks.

## Structure

- `data/`: Contains project metadata and catalog information.
- `outputs/`: Contains the actual outputs generated by the models.
  - `modern-landing-page/`: HTML landing pages.
  - `logic-puzzle/`: Text files with puzzle solutions.
  - `todo-app/`: Functional todo applications.
